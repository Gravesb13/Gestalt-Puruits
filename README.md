# Gestalt-Pursuits
Pursuits are a new optional feat category that grants characters access to features from another class. Unlike standard feats, each Pursuit is a four-tier investment — representing the slow, earned mastery of a second discipline over the course of an adventuring career.

Each feat is tied to a specific class (Barbarian, Bard, Cleric, etc.) and grants that class's most iconic features in a scaled, balanced form. A Fighter who invests in the Wizard feat doesn't become a Wizard — but by level 20, they will cast spells in a way that feels meaningfully arcane.

Pursuits are homebrew optional rules and are not part of the official 2024 D&D rules. All features within them are designed to complement the 2024 Player's Handbook rules. DM approval is recommended before using these feats, and some higher-tier subclass features may require additional table discussion.
